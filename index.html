Semantic Sabotage is an open source platform for creating live typographic YouTube remixes.


Each remix uses HTML5 and Javascript to visualize the transcript of a YouTube movie, as it plays. Big thanks to James Pennebaker and the UT Austin Department of Psychology, whose research made this project possible.

Download the code from GitHub and create your own mutation. Start with one of the templates or with an existing remixes.
